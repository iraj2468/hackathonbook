# Glossary

This section contains definitions of key terms used throughout the textbook.

## Terms

*   **ROS 2**: Robot Operating System 2
*   **Gazebo**: A 3D robot simulator
*   **NVIDIA Isaac Sim**: A scalable robotics simulation application and synthetic data generation tool
*   **VLA Systems**: Vision-Language-Action Systems
*   **Humanoid Robot**: A robot with a body shape built to resemble the human body
